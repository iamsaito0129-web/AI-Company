# 冬の北海道旅行パッケージ

## 概要
- **基本の日数**: 3泊4日
- **追加プラン (+n日)**: 小樽・砂川方面を1日かけてゆっくり巡る (+1日)

## タイムライン（基本：3泊4日）
### Day 1: 札幌到着と都市型観光
- [ ] 08:30 伊丹空港出発（ANA0771便）
- [ ] 10:20 新千歳空港到着、JR快速エアポートで札幌へ
- [ ] 12:00 札幌駅周辺で海鮮ランチ
- [ ] 16:00 AOAO SAPPORO（都市型水族館）で見学
- [ ] 19:00 すすきのでジンギスカンまたはスープカレーの夕食

### Day 2: 圧巻のパウダースノー・スキー
- [ ] 08:07 真駒内駅からバスで札幌国際スキー場へ
- [ ] 09:00〜16:00 札幌国際スキー場でスキー（手ぶらセット活用）
- [ ] 17:10 帰路バス出発、札幌市内へ
- [ ] 夜 ホテル大浴場でリフレッシュ

### Day 3: 小樽運河と聖地巡礼
- [ ] 09:00 二条市場で海鮮丼の朝食
- [ ] 11:00 小樽運河散策、または「水曜どうでしょう」聖地の平岸高台公園へ
- [ ] 19:00 札幌市内でまだ食べていないマストグルメを堪能

### Day 4: 最新の球場体験と帰路
- [ ] 12:30 エスコンフィールドHOKKAIDOでスタジアムツアー参加
- [ ] 14:00 新千歳空港へ移動、ラーメン道場やお土産探し
- [ ] 16:15 新千歳空港出発（ANA0778便）

## 追加オプション (+1日)
### Day 5: 小樽・砂川深掘り
- [ ] 小樽堺町通りでスイーツ食べ歩き（ルタオ本店）
- [ ] SHIRO砂川本店で香りの体験
- [ ] JRタワーT38から札幌のパノラマ夜景

## 持ち物・準備
- [ ] 航空券（eチケット）
- [ ] スキー装備（グローブ、ニット帽、厚手ソックス、ネックウォーマー）
- [ ] 防寒インナー上下（ヒートテック等）
- [ ] 折りたたみ傘・滑り止め付きの靴

## 現実逃避ポイント（癒やし・驚き）
- 札幌国際スキー場の圧倒的な雪質「パウダースノー」。
- エスコンフィールドHOKKAIDOの最新鋭設備と開放感。
- 凍れる空気の中で食べる熱々の味噌ラーメンやスープカレー。
