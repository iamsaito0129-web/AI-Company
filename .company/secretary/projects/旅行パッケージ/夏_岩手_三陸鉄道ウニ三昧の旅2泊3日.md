# 夏の岩手・三陸鉄道ウニ三昧の旅2泊3日

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: 盛岡わんこそば体験と、小岩井農場で自然を満喫 (+1日)

## タイムライン（基本：2泊3日）
### Day 1: 盛岡・三陸の玄関口
- [ ] 11:30 盛岡駅到着。盛岡冷麺で腹ごしらえ。
- [ ] 13:00 バス、またはJR山田線で宮古へ移動。
- [ ] 15:30 浄土ヶ浜散策。青い海と白い岩のコントラスト。
- [ ] 18:00 三陸の海の幸を堪能。宿泊。

### Day 2: 三陸鉄道・走破と絶景
- [ ] 09:00 宮古駅から三陸鉄道リアス線（北リアス線）に乗車。
- [ ] 11:00 絶景ポイント（安家川橋梁など）で車窓を楽しむ。
- [ ] 12:30 久慈駅到着。お待ちかねの「うに丼」・「まめぶ汁」お昼ごはん。
- [ ] 15:00 小袖海岸（あまちゃん聖地）散策、釣り堀体験等。
- [ ] 18:00 久慈、または八戸方面へ移動し、一泊。

### Day 3: 地底湖と帰路
- [ ] 09:30 日本三大鍾乳洞、龍泉洞へ。青い「ドラゴンブルー」の地底湖。
- [ ] 13:00 岩手めんこいテレビゆかりの地、または盛岡市内観光。
- [ ] 16:00 帰路へ。

## 追加オプション (+1日)
### Day 4: 盛岡・自然体験
- [ ] 盛岡で「わんこそば」に挑戦。限界を知る。
- [ ] 小岩井農場でバター作り体験、動物とふれあい。

## 持ち物・準備
- [ ] 青春18きっぷ（時期による、三鉄は別途）
- [ ] 上着（鍾乳洞内は夏でも10度前後）
- [ ] カメラ（三鉄の絶景を激写）

## 現実逃避ポイント（癒やし・驚き）
- 三陸鉄道の窓から見る、どこまでも続く水平線。
- 透き通る龍泉洞の底まで見えるブルー。
- 獲れたてウニ。至福の瞬間。
