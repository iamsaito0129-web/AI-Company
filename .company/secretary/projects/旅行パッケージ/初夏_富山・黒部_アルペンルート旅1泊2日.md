# 初夏の富山・黒部アルペンルート旅1泊2日

## 概要
- **基本の日数**: 1泊2日
- **追加プラン (+n日)**: 黒部ダム周辺のハイキング、または欅平まで行く黒部峡谷鉄道トロッコ列車の旅 (+1日)

## タイムライン（基本：1泊2日）
### Day 1: 富山到着・黒部ダムの絶景へ
- [ ] 08:30 JR富山駅集合、電鉄富山駅へ移動
- [ ] 09:00 立山黒部アルペンルート開始（立山駅〜美女平〜弥陀ヶ原）
- [ ] 12:00 室堂にてランチ（標高2,450mの絶景）
- [ ] 13:30 雪の大谷ウォーク（6月上旬まで）を堪能
- [ ] 15:00 黒部ダム到着、放水の迫力を楽しむ
- [ ] 18:00 扇沢駅へ抜け、富山市内のホテルに戻る
- [ ] 20:00 富山駅周辺で「富山湾の宝石」白えび・ホタルイカを堪能

### Day 2: 城下町・水の都の散策
- [ ] 09:30 富岩運河環水公園へ、世界一美しいスタバを堪能
- [ ] 11:30 富山市ガラス美術館で芸術に触れる
- [ ] 13:00 ランチ：富山ブラックラーメン、または鱒の寿し
- [ ] 15:00 帰路へ

## 追加オプション (+1日)
### Day 3: トロッコ列車で深淵な黒部へ
- [ ] 黒部市・宇奈月温泉へ移動
- [ ] 黒部峡谷鉄道トロッコ列車に乗車
- [ ] 欅平でのんびり足湯と秘境探索

## 持ち物・準備
- [ ] 高地用防寒着（夏でも室堂は気温10度以下の可能性あり）
- [ ] サングラス（雪の照り返し対策）
- [ ] 歩きやすい靴（スニーカー、登山靴）

## 現実逃避ポイント（癒やし・驚き）
- 雪の大谷の巨大な雪壁。日本とは思えないスケール感。
- 黒部ダム放水の圧倒的な水量と迫力。
- 立山連峰のパノラマビュー。都会の雑踏をリセット。
