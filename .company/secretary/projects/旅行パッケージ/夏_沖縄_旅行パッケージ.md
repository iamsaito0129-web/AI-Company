# 夏の沖縄旅行パッケージ

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: 北部滞在を延ばして、美ら海水族館や今帰仁城跡を巡る (+1日)

## タイムライン（基本：2泊3日）
### Day 1: 到着と青の洞窟アクティビティ
- [ ] 07:00 関西国際空港集合（時間厳守！）
- [ ] 10:10 那覇空港到着、レンタカー受け取り
- [ ] 12:00 瀬長島ウミカジテラスで海を眺めながらランチ
- [ ] 15:30 恩納村・青の洞窟でダイビング体験（感動のブルー）
- [ ] 20:00 那覇市内ホテルチェックイン、国際通りで夕食・散策

### Day 2: 北部の自然とオリオンビール
- [ ] 09:00 ホテル出発、沖縄自動車道で一気に北部へ
- [ ] 11:30 アスムイハイクス（大石林山）で聖地ハイキング・絶景堪能
- [ ] 14:30 オリオンハッピーパークで工場見学（出来立てビールの試飲）
- [ ] 17:30 北谷・アメリカンビレッジでサンセットを見ながらディナー
- [ ] 21:00 ホテルへ戻る

### Day 3: 学びと帰路
- [ ] 09:00 チェックアウト、荷物を車へ
- [ ] 10:00 琉球大学キャンパス散策（ローカルな雰囲気）
- [ ] 12:00 那覇周辺で沖縄そばのランチ
- [ ] 13:30 レンタカー返却、那覇空港へ
- [ ] 19:55 那覇空港出発（ANA1740便）

## 追加オプション (+1日)
### Day 4: やんばるの森深掘り
- [ ] 美ら海水族館でジンベエザメ鑑賞
- [ ] 備瀬のフクギ並木でサイクリング
- [ ] 古宇利島でハートロック観光

## 持ち物・準備
- [ ] 水着・ラッシュガード（ダイビング用）
- [ ] サンダル・歩きやすい靴（大石林山用）
- [ ] 免許証（レンタカー必須）
- [ ] 日焼け止め・サングラス

## 現実逃避ポイント（癒やし・驚き）
- 青の洞窟の圧倒的な透明感。
- 大石林山の奇岩群とガジュマルの巨木が生み出す原始的な風景。
- アメリカンビレッジの異国情緒溢れる街並み。
