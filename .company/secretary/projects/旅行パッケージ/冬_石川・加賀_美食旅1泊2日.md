# 冬の石川・加賀美食旅1泊2日

## 概要
- **基本の日数**: 1泊2日
- **追加プラン (+n日)**: 能登島までドライブし、のとじま水族館や七尾湾を巡る (+1日)

## タイムライン（基本：1泊2日）
### Day 1: 金沢美食・アート・温泉
- [ ] 10:00 金沢駅・鼓門で記念撮影。
- [ ] 11:00 近江町市場で海鮮丼ランチ。冬の味覚「カニ・エビ」満載。
- [ ] 13:00 兼六園で冬の風物詩「雪吊り」を観賞。
- [ ] 15:00 金沢21世紀美術館で最新アート体験。
- [ ] 17:00 加賀温泉郷（山しろ・山なか）の旅館へ。
- [ ] 19:00 お待ちかねのカニフルコース夕食。

### Day 2: 温泉街散策とレトロな風景
- [ ] 09:30 温泉街をゆったり散策。山中温泉ゆげ街道で食べ歩き。
- [ ] 11:00 鶴仙渓散策、冬の凛とした空気。
- [ ] 13:00 お茶屋遊び体験、または九谷焼の絵付け体験。
- [ ] 15:00 お土産（金箔、地酒、九谷焼）を購入し、帰路へ。

## 追加オプション (+1日)
### Day 3: 能登の海岸線
- [ ] 千枚田のライトアップ（あぜのきらめき）を見学。
- [ ] 能登島水族館。
- [ ] ひがし茶屋街で抹茶と和菓子。

## 持ち物・準備
- [ ] 暖かい防寒着、防水の靴（雪予報の場合）
- [ ] 旅館を楽しむためのリラックスウェア（自前）

## 現実逃避ポイント（癒やし・驚き）
- 兼六園の幾何学的な雪吊りの美しさ。
- カニ一色！至極の夕食タイム。
- 立ち込める湯けむりと、雪国情緒あふれる街歩き。
