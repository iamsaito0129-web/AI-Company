# 🌸 新生活準備：小学校教員・公務員としての立ち上げ

## 🔑 役所手続き・書類 (3月下旬〜4月1日)
- [ ] **教員免許状の原本確認**
    - [ ] 提出が必要な場合が多いため、原本をすぐに取り出せる場所に整理。写しも数枚取っておく。
- [ ] **卒業証明書・修了証明書（原本）**
    - [ ] 最終学歴の証明として必要。
- [ ] **職歴証明書（前職がある場合）**
    - [ ] 給与の号給算定に不可欠なため、以前の職場に依頼済みか確認。
- [ ] **健康診断証明書（指定項目）**
    - [ ] 雇入時の健康診断結果。原本提出が必要な場合が多い。
- [ ] **マイナンバーカードの写し**
    - [ ] 扶養控除申告書等で使用。
- [ ] **給与振込口座の通帳・印鑑**
    - [ ] 自治体指定の銀行口座（例：大阪府なら三菱UFJ/三井住友/ゆうちょ等）を準備。
- [ ] **通勤定期・経路申請（経路図作成）**
    - [ ] 最安経路の運賃を調べ、手書きまたはデジタルでの地図準備。
- [ ] **身務書類用の印鑑セット**
    - [ ] **認印（朱肉用）**: 行政文書の決裁用。100均以外で1本しっかりしたものがあると良い。
    - [ ] **訂正印**: 5mm程度の小さいもの。
    - [ ] **シャチハタ（データネーム）**: 日付が入るタイプが連絡帳や校内配布物の確認に非常に便利。
- [ ] **年金手帳・各種控除申告書**
    - [ ] 4月1日の会合で一斉に書かされるため、家族の生年月日などの情報をメモ。

## 🏫 学校現場での必須アセット（ベテラン推奨）
- [ ] **服装・身の回り**
    - [ ] **スーツ (2着以上)**: 着任式、入学式用。一着は「ロッカー置き」しておくと不意の来客時も安心。
    - [ ] **ジャージ・体育着 (2セット以上)**: 運動中に入れ替えるため。
    - [ ] **外用運動靴**: 泥汚れしやすいので、予備を含め。
    - [ ] **内履き (職員室用)**: 疲れにくいナースサンダルやオフィスシューズ。
    - [ ] **体育館シューズ**: 指定がない限り、動きやすさ重視のもの。
- [ ] **校務・授業効率化グッズ**
    - [ ] **キッチンタイマー (複数個)**: 授業の区切り、給食残り、自習時間など並行して使う。
    - [ ] **ホイッスル & 拍子木・リズム太鼓**: 注意を一瞬で向けるための音響ツール。
    - [ ] **チョークホルダー**: 手が汚れず、チョークの折れを防ぐ（シンワ製等が人気）。
    - [ ] **ミニホワイトボード & マグネット**: 黒板の脇で掲示や個別の指示出しに。
    - [ ] **「がんばり賞・御褒美」スタンプ**: 採点のスピードアップと児童のモチベーション向上。
- [ ] **文房具（厳選）**
    - [ ] **PILOT Vコーン（赤・黒）**: 水性で裏写りしにくく、採点に最適。
    - [ ] **15cm・30cm定規**: 事務作業と授業用。
    - [ ] **マスキングテープ**: 掲示物の貼り替えに。
- [ ] **衛生・飲食**
    - [ ] **水筒 & マグカップ**: 職員室での「ホッと一息」用。
    - [ ] **給食セット**: エプロン、ハンカチ、マスク予備、箸。
    - [ ] **ハンドタオル (3〜5枚)**: 1日の中で取り替えることが多い。

## 🍎 戦略的「黄金の3日間」シナリオ
- [ ] **1日目：出会いと安心感の構築**
    - [ ] **自己紹介クイズ**: 自分の好きなもの、家族、先生になった夢などを○×クイズに。
    - [ ] **これだけは守る約束（鉄則）**: 「命に関わること」や「人の心を傷つけること」など、最重要ルールを1つ。
- [ ] **2日目：ルール作りと交流**
    - [ ] **バースデーライン**: しゃべらずに誕生日順に並ぶゲーム。協力の楽しさを。
    - [ ] **学級のめあて（キーワード）**: 児童から「どんなクラスにしたいか」を吸い上げる。
- [ ] **3日目：ルーティンの確立**
    - [ ] **給食・清掃のデモンストレーション**: 実際に動いて見せ、成功体験を積ませる。
