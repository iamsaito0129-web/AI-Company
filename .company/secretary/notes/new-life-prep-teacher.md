# 🌸 新生活準備：小学校教員・公務員としての立ち上げ

## 🔑 役所手続き・書類 (3月中に完了)
- [ ] **教員免許状の原本確認**
    - [ ] 提出が必要な場合が多いため、すぐに取り出せる場所に整理。
- [ ] **健康診断証明書**
    - [ ] 指定された項目の診断が済んでいるか、手元に写しがあるか確認。
- [ ] **通勤定期・経路申請**
    - [ ] 最安・最短経路を把握し、申請書類の書き方を予習（公務員は厳格です）。
- [ ] **身務書類の印鑑**
    - [ ] **認印（朱肉用）**: 予備を含め2本あると安心（学校と自宅、等）。
    - [ ] **シャチハタ**: 記録簿などの日常的な確認用。

## 🏫 学校現場での必須アセット
- [ ] **上履き（サンダル不可）**
    - [ ] 非常時に児童を誘導できるよう、かかとのある動きやすいもの。
- [ ] **体育館シューズ**
    - [ ] 体育の授業に即応できるよう、上履きとは別に用意。
- [ ] **ジャージ・体育着**
    - [ ] 1日中スーツではなく、着替える場面が多いため必須。
- [ ] **筆記用具・校務用具**
    - [ ] 赤ペン（採点用）、タイマー（授業時間管理用）、ホイッスル（体育・外遊び用）。

## 🍎 4月の「学級開き」に向けたアイデア
- [ ] **自己紹介の準備**
    - [ ] **3分間トーク:** 自分の好きなこと、先生になった理由（3年生〜高学年向け）。
    - [ ] **クイズ形式:** 自分の名前や趣味をクイズにして児童に当てる（低学年〜中学年向け）。
- [ ] **学級目標の案**
    - [ ] どんなクラスにしたいか、キーワードを3つほど考えておく。

## 💡 公務員としての生活リズム (朝型への移行)
- [ ] **7:30〜7:45 出勤のリズム**
    - [ ] 学校の朝は早いため、今のうちからこの時間に活動を開始する習慣。
