# プロジェクト: パズル解決プロジェクト

## 概要
3D空間内の面を移動するルートパズルを自動で解決するためのアルゴリズム開発。規定の条件（特定の面をすべて通過、壁を回避、手足の制約など）を満たす最短または有効なルートを算出する。

## ゴール
- [x] 5x5x10のグリッドにおける面移動のモデリング
- [x] 全経由地（イエロー面）を網羅するDFS（深さ優先探索）の実装
- [x] 解決ルートの可視化（HTML出力）
- [x] 特定の制約（16ターンの制約など）を満たす解の特定
