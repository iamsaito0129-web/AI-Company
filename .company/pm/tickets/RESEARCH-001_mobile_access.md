# 調査チケット: [RESEARCH-001] モバイル操作向けのセキュアなリモートアクセス手法の調査

## 対象プロジェクト
- Mobile Agent Controller

## 調査目的
スマホからPC上の antigravity エージェントを「安全」かつ「簡単」に操作するための最適なインフラ・プロトコルを特定する。

## 候補案
1. **Tailscale (Mesh VPN) + SSH/WebUI**
   - メリット: セキュリティが極めて高い、設定が容易。
   - 懸念: スマホ側のクライアントアプリが必要。
2. **Cloudflare Tunnel + Zero Trust 認証**
   - メリット: ブラウザだけで完結できる、独自ドメインが使える。
   - 懸念: 設定がやや複雑。
3. **ngrok / VS Code Remote Tunnels**
   - メリット: 設定が最も速い。
   - 懸念: 永続的な利用や詳細なカスタマイズ性。
