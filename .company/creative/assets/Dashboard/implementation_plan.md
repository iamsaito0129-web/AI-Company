# 資産管理 Dashboard Premium - Implementation Plan

## 1. 概要
ユーザーの持つ Notion 資産管理 DB (支出、収入、決済) を統合し、視覚的に美しく、管理しやすい Web ダッシュボードを提供します。

## 2. 独自カスタマイズのキモ
- **馬渕個別特化型計算**: 85分ごとのコマ数管理と給与概算機能の搭載。
- **純資産の可視化**: 複数の口座や電子マネーを合算した真の純資産をリアルタイム表示。
- **Premium Aesthetics**: グラスモーフィズムと 4K 高級画像背景による「見たくなる」デザイン。

## 3. 実装フェーズ
- [x] Phase 1: モックアップ UI の構築 (現在の状態)
- [ ] Phase 2: Notion API との権限疎通 (ユーザー承認待ち)
- [ ] Phase 3: 自動給与計算ロジックの実装
- [ ] Phase 4: モバイルレスポンシブ最適化
