# ワークフロー: デイリー・シンク（朝会）

## 目的
全エージェントの状態を同期し、今日の目標を明確にする。

## 手順
1.  **秘書**: `03_OPERATIONS/todos/` から前日の未完了タスクと今日の予定を抽出する。
2.  **全部門**: 現在のボトルネックや困りごとを報告する。
3.  **CEO**: 報告に基づき、今日の優先順位を決定し、指示を出す。
4.  **秘書**: 決定事項を今日のTODO（アーティファクト）として出力する。
5.  **全部門**: マネージャービューで並列作業を開始する。

## トリガー
- 毎朝のセッション開始時。
- 「朝会を開始して」「状況を同期して」という指示。
