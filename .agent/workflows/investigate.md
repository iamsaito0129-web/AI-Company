// turbo-all
1. 調査対象の特定
   - ユーザーからのインプットに基づき、検索キーワードを選定。

2. browser_subagent による自律調査
   - 関連サイトの巡回。
   - 重要な画面のスクリーンショット撮影。
   - 複雑な挙動の録画。

3. レポート作成
   - `.company/research/topics/[テーマ].md` に保存。
   - スクリーンショットと録画へのリンクを含める。
   - 比較表や結論を明文化。

4. 報告
   - 完成したレポートを Artifacts として提示。
