# 春秋の愛媛・赤髪のとも聖地巡礼2泊3日

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: 小豆島や香川方面へ渡り、うどん巡りを追加 (+1日)

## タイムライン（基本：2泊3日）
### Day 1: 松山到着・道後温泉とみかん三昧
- [ ] 10:30 松山空港到着。「愛媛の食卓1970」で蛇口からみかんジュースを体験。
- [ ] 12:00 ランチ：太田屋、または「丸ぶん」で鯛めし（北条鯛めし/宇和島鯛めし）を堪能。
- [ ] 14:00 道後温泉散策。道後温泉本館、坊っちゃんカラクリ時計を見学。
- [ ] 16:00 10FACTORYでみかんジュースの飲み比べ。
- [ ] 18:00 道後プリンスホテル、または周辺の旅館にチェックイン。温泉でリラックス。

### Day 2: しまなみ海道サイクリングと絶景
- [ ] 09:00 今治へ移動。しまなみ海道サイクリング開始。
- [ ] 11:30 亀老山展望公園から来島海峡大橋の絶景を眺める。
- [ ] 13:00 道の駅 今治湯ノ浦温泉でランチ。
- [ ] 15:30 松山へ戻る道中、ふたみシーサイド公園で休憩。
- [ ] 17:00 日本一海に近い駅「下灘駅」でサンセットを鑑賞（動画の名シーン）。

### Day 3: 松山城とレトロ散策
- [ ] 09:30 松山城をロープウェイで訪問。天守閣からの眺望を楽しむ。
- [ ] 12:00 萬翠荘、または大街道周辺で最後の愛媛グルメ。
- [ ] 14:00 絣屋本店でお土産選び。
- [ ] 16:00 松山空港より帰路へ。

## 追加オプション (+1日)
### Day 4: 瀬戸内横断
- [ ] フェリーで広島、または香川へ移動。
- [ ] 尾道の坂道散策、または金刀比羅宮参拝。

## 持ち物・準備
- [ ] 動きやすい服・スニーカー（サイクリング・お城・階段用）
- [ ] カメラ（下灘駅、亀老山は撮影必須）
- [ ] タオル（道後温泉外湯巡り用）

## 現実逃避ポイント（癒やし・驚き）
- 下灘駅のホームから眺める、空と海が溶け合うような夕景。
- 蛇口からジュースが出る、大人も子供もワクワクする体験。
- しまなみ海道を自転車で駆け抜ける爽快感。
