# 愛媛の赤髪のとも聖地巡礼・2泊3日旅行パッケージ

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: サイクリング（しまなみ海道）特化プラン

## タイムライン（基本：2泊3日）
### Day 1: 松山市内と文学の風
- [ ] 松山空港/松山駅到着
- [ ] **松山城**: ロープウェイで登閣し、市内を一望。
- [ ] **道後温泉本館**: 日本最古の温泉で癒やしの時間。
- [ ] 赤髪のともさんが訪れたグルメスポットでの夕食。

### Day 2: 絶景と癒やしの伊予路
- [ ] **下灘駅**: 「日本一海に近い駅」として有名な絶景スポットで記念撮影。
- [ ] **大洲の町並み**: 「伊予の小京都」を散策。
- [ ] **内子**: 歴史的な街並みと木蝋の歴史に触れる。

### Day 3: 海の恵みとアクティビティ
- [ ] **しまなみ海道**: 短距離のサイクリング体験、または来島海峡急流観潮船。
- [ ] 蛇口からみかんジュース体験（空港や観光施設）。
- [ ] お土産購入（今治タオル、タルト等）。
- [ ] 松山出発

## 追加オプション (+1日)
### Day 4: しまなみ海道フル走破
- [ ] 今治から尾道へ向けての本格サイクリング。

## 持ち物・準備
- [ ] 歩きやすい靴（城や街歩き多め）
- [ ] 日焼け止め（瀬戸内の陽光は強いです）
- [ ] カメラ（下灘駅は必須）

## 現実逃避ポイント（癒やし・驚き）
- 下灘駅のベンチで、ただ海を眺めるだけの贅沢な時間。
- 蛇口から出てくる、濃厚な愛媛みかんジュースの驚き。
- 道後温泉の歴史が醸し出す、非日常的な空間。
