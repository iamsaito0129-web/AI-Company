# 春秋の岡山・香川アート巡り2泊3日

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: 高松周辺のうどん巡りを極める (+1日)

## タイムライン（基本：2泊3日）
### Day 1: 倉敷美観地区・レトロ散策
- [ ] 11:00 倉敷美観地区到着。白壁の街並みをゆったり散策。
- [ ] 12:30 大原美術館で見事なエル・グレコの名画を。
- [ ] 15:00 アイビースクエアで体験、または倉敷帆布などの雑貨巡り。
- [ ] 18:00 瀬戸大橋線を経由して高松市内へ。名物「骨付鳥」で乾杯。

### Day 2: 直島・アートの島
- [ ] 08:00 高松港よりフェリーで直島へ。
- [ ] 10:00 地中美術館、李禹煥美術館。建築と自然の融合を。
- [ ] 13:00 家プロジェクト散策。島民の暮らしとアートの調和。
- [ ] 16:00 赤かぼちゃ、黄かぼちゃで記念撮影。
- [ ] 18:00 高松に戻り、宿泊。

### Day 3: 金刀比羅宮・うどん・帰路
- [ ] 09:00 こんぴらさん（金刀比羅宮）参拝。1368段の階段に挑戦。
- [ ] 12:00 もちろん手打ちうどん！名店をハシゴ。
- [ ] 14:00 栗林公園で見事な大名庭園を鑑賞。
- [ ] 16:00 高松駅・高松空港より帰路へ。

## 追加オプション (+1日)
### Day 4: 豊島・小豆島
- [ ] 豊島美術館。ただそこに居るだけで心が洗われる場所。
- [ ] 小豆島オリーブ公園、エンジェルロード。

## 持ち物・準備
- [ ] 歩きやすい靴（こんぴらさん、直島散策用）
- [ ] 日焼け止め（海辺・島巡り用）
- [ ] 御朱印帳

## 現実逃避ポイント（癒やし・驚き）
- フェリーから眺める穏やかな瀬戸内海。
- 建築家・安藤忠雄氏による、光を操る美術館。
- こんぴらさん参拝後の達成感と、冷たいうどんの一杯。
