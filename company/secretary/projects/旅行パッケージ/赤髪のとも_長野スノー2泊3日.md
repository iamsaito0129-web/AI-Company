# 長野の赤髪のとも（ともズズ）2名旅・スノー＆アニマルパッケージ

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: スキー・スノーボード1日追加

## タイムライン（基本：2泊3日）
### Day 1: 長野の冬を味わう
- [ ] 長野駅到着。
- [ ] 善光寺参拝と仲見世通りでの食べ歩き。
- [ ] 渋温泉、または湯田中温泉へ移動・宿泊。

### Day 2: スノーモンキーと冬の絶景
- [ ] **地獄谷野猿公苑**: 温泉に入る猿（スノーモンキー）を観察。雪道を歩くので準備必須。
- [ ] スキー場周辺へ移動、または温泉街でゆっくり。
- [ ] 地元の信州牛や山菜料理を堪能。

### Day 3: スノーアクティビティ
- [ ] **北志賀・志賀高原エリア**: スノーフィールドでアクティビティ（そり遊び、スノーシュー等）。
- [ ] お土産購入（おやき、リンゴ菓子等）。
- [ ] 長野発

## 追加オプション (+1日)
### Day 4: 本格ウィンタースポーツ
- [ ] 志賀高原や竜王スキーパーク等の広大なゲレンデで終日滑走。

## 持ち物・準備
- [ ] スノーブーツ（野猿公苑への道は凍結の恐れあり）
- [ ] 防寒着（マイナス気温を想定）
- [ ] スマホ防水ケース（雪山での撮影用）

## 現実逃避ポイント（癒やし・驚き）
- 湯煙の中で気持ちよさそうに目を細める猿たちの表情。
- 雪景色を眺めながら入る、極上の露天風呂。
- 友人との何気ない会話が弾む、静かな温泉街の夜。
