# 秋の青森・奥入瀬十和田湖ツアー2泊3日

## 概要
- **基本の日数**: 2泊3日
- **追加プラン (+n日)**: 下北半島・大間まで足を伸ばし、マグロを堪能する旅 (+1日)

## タイムライン（基本：2泊3日）
### Day 1: 青森空港・ねぶた・美食
- [ ] 10:00 青森空港到着。レンタカー借受。
- [ ] 11:00 青森魚菜センター（古川市場）で「のっけ丼」ランチ
- [ ] 13:30 ねぶたの家 ワ・ラッセで迫力のねぶたを見学
- [ ] 15:00 A-FACTORYでお土産とシードル試飲
- [ ] 18:00 青森市内のホテルへ。夜は地元の居酒屋で十和田バラ焼き。

### Day 2: 奥入瀬渓流ハイキング
- [ ] 09:00 十和田湖・休屋へ向け出発
- [ ] 11:00 奥入瀬渓流散策。石ヶ戸〜銚子大滝をゆったり歩く。
- [ ] 13:00 奥入瀬渓流ホテル、または周辺でランチ。
- [ ] 15:00 十和田湖遊覧船に乗車。湖上から紅葉を眺める。
- [ ] 18:00 旅館で温泉。秋の味覚満載の夕食。

### Day 3: 八甲田の紅葉と帰路
- [ ] 09:00 酸ヶ湯温泉にて千人風呂(混浴注意)を体験、または蔦沼散策。
- [ ] 11:00 八甲田ロープウェーで山頂公園へ。360度の紅葉パノラマ。
- [ ] 13:00 青森市内に戻り、煮干しラーメンで締める。
- [ ] 15:00 レンタカー返却、青森空港へ。

## 追加オプション (+1日)
### Day 4: 本州北端のドラマ
- [ ] 尻屋崎の寒立馬を見学。
- [ ] 大間崎で念願の「大間のマグロ」を食す。
- [ ] 仏ヶ浦の奇岩地帯。

## 持ち物・準備
- [ ] 重ね着（秋の青森は急激に冷え込みます）
- [ ] 散策用の歩きやすい靴
- [ ] カメラの予備バッテリー（紅葉を撮りすぎます）

## 現実逃避ポイント（癒やし・驚き）
- 奥入瀬のせせらぎと苔、紅葉のコラボレーション。
- 八甲田ロープウェーから見下ろす極彩色の森。
- 都会の疲れが吹き飛ぶ、温泉と新鮮な魚介類のパワー。
