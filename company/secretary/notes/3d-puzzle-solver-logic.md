# 🧩 3Dパズル解決プロジェクト：進捗・ナレッジノート

## 1. プロジェクト概要
直方体（5x5x10）の表面を移動し、特定の黄色タイルをすべて通過し、16ターン以内でゴールに到達するルートを探索する。

## 2. 実装済み重要ロジック
### 物理隣接関係 (`get_adj_v3`)
- **面境界の移動**: 面Aから面Bへの移動を、3D座標の幾何学的関係に基づき正確に判定。
- **カド曲がり**: 隣接する面への移動だけでなく、角（頂点）を共有する面への斜め移動に近い「面切り替え」を定義。
- **17番（カド）**: 渦巻きタイルの上で移動が発生する場合、物理的な方向転換を伴うため「1ターン」としてカウントする。

### 探索アルゴリズム
- **DFS (深層優先探索)**: ターン数制限と全黄色マス通過を制約条件として実装。
- **高速化**: 各ステップでの訪問状態を `bitmask` で管理し、重複探索を排除。

## 3. IDマッピングの同期
PythonソルバーとHTML（展開図）の不整合を解消。
- **展開図の補正**: 上蓋・下蓋の5列オフセットを考慮したID生成ロジック。
- **座標同期**: 233番と228番が物理的に隣接しない（233は `(4, 4, 1, 2)`, 228は `(4, 3, 9, 4)`）ことを確認し、ソルバーの厳密な物理モデルに反映。

## 4. 未解決・今後の課題
- **解の発見**: 現在の物理制約条件下で、16ターン以内に全黄色マスを網羅するルートが非常に厳しく、探索を継続中。
- **ヒューリスティック改善**: マンハッタン距離をベースにした枝刈りロジックのさらなる強化。
