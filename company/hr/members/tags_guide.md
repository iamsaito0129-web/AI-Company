# 🏷️ メンバー管理タグ・ガイドライン

メンバープロフィールのフロントマターで使用するタグの定義集です。これらを活用することで、Obsidian内でのフィルタリングや、Dataviewによる動的なリスト生成が可能になります。

## 1. 関係性・属性 (`#rel/`)
「その人は誰か？」という基本的な属性を定義します。

| タグ | 定義 |
| :--- | :--- |
| `#rel/family` | 家族（直系） |
| `#rel/relative` | 親戚 |
| `#rel/friend` | 友人一般 |
| `#rel/bestie` | 親友・特に親しい |
| `#rel/junior` | 後輩 |
| `#rel/senior` | 先輩 |
| `#rel/colleague` | 仕事関係・教員仲間 |
| `#rel/student` | 教え子・学生 |

## 2. コミュニティ・所属 (`#org/`)
「どこで繋がっているか？」という所属を定義します。

| タグ | 定義 |
| :--- | :--- |
| `#org/univ` | 大学関連の繋がり |
| `#org/festival-staff` | 大学祭実行委員会のメンバー |
| `#org/local-event` | たこづくり大会など地域活動の仲間 |
| `#org/school` | 学校・職場の同僚 |
| `#org/childhood` | 幼馴染・地元の友人 |

## 3. コンタクト状態 (`#status/`)
「今どういう距離感か？」という状態を定義します。

| タグ | 定義 |
| :--- | :--- |
| `#status/vip` | 最重要人物（頻繁に連絡を取る） |
| `#status/active` | 現在進行系でやり取りがある |
| `#status/dormant` | 以前は交流があったが今は疎遠（休眠状態） |
| `#status/lost-touch` | 連絡先不明・完全に疎遠 |

## 4. 興味・関心・専門性 (`#interest/`)
「何に詳しいか・何が好きか？」を定義します。ギフト選びや相談時に活用します。

| タグ | 定義 |
| :--- | :--- |
| `#interest/tech` | IT・最新ガジェット・プログラミング |
| `#interest/edu` | 教育・授業実践・学校経営 |
| `#interest/art` | デザイン・写真・アート・クリエイティブ |
| `#interest/foodie` | グルメ・食べ歩き・料理 |
| `#interest/liquor` | お酒好き（ワイン、日本酒、ビール等） |
| `#interest/sports` | スポーツ・アウトドア・健康 |

## 5. システム運用 (`#type/`)
AIエージェントによる管理やメンテナンス用のタグです。

| タグ | 定義 |
| :--- | :--- |
| `#type/input-pending` | プロフィールの詳細が未入力の状態 |
| `#type/needs-catchup` | 一定期間連絡がなく、近況確認が推奨される人 |
| `#type/grad-thesis` | 卒論協力など、特定の限定的なプロジェクトに関連 |

---
> [!TIP]
> タグを付与する際は、複数を組み合わせる（例：`#rel/junior` 且つ `#org/festival-staff`）ことで、より精度の高い検索が可能になります。
