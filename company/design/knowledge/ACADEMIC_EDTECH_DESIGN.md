# DESIGN.md - アカデミック・エドテックスタイル

## 1. デザインの基本理念 (Visual Strategy / Principles)
- **コンセプト:** 「知の探検（Intellectual Expedition）」 - 難しい概念を親しみやすく、かつ論理的に整理して提示。
- **ターゲット:** 中学生、高校生、および教育専門家。
- **トーン＆マナー:** 知的、整理された、ユーモアのある、安心感。

## 2. カラーパレット (Color Palette)
- **プライマリーカラー:** セージグリーン (`#C2E5D3`) - 思考を助け、目の疲れを和らげる。
- **アクセントカラー:** テラコッタ・クレイ (`#E07A5F`) - 重要なポイントや「！」の喚起。
- **セマンティックカラー:** 
  - Solution: ディープグリーン (`#2D6A4F`)
  - Warning: ピーチイエロー (`#F4A261`)
- **ニュートラルカラー:** 
  - Page Background: ミントホワイト (`#F0F7F4`)
  - Header Text: フォレストブラック (`#1B4332`)

## 3. タイポグラフィ (Typography)
- **フォントファミリー:**
  - 和文: Noto Sans JP (Medium) / 筑紫A丸ゴシック (TsukuARdGothic)
  - 英文: Roboto Serif / Computer Modern Font (数式)
- **階層（Hierarchy）:**
  - Main Title: 40px, Bold, 黒の細いアウトライン。
  - Subsection: 20px, Semi-Bold, カプセル型背景（Badge）。
  - Body: 16px, Regular, 行間 1.7.

## 4. コンポーネントのルール (Components & UI Elements)
- **マスコット:** 偉人、または幾何学図形をモチーフにした親しみやすいキャラクター（例：デフォルメされたピタゴラス）。
- **数式・図形:** 黒の 1px - 2px の手書き風の線（Dashed lines）を使用。透明度の高い網掛けを活用。
- **インフォメーション:** 吹き出し（Tooltip Style）と矢印を用いた視線誘導。

## 5. インタラクションとアニメーション (Interactions & Motion)
- **トランジション:** スライドイン（左から右）による論理的展開の表現。
- **マイクロインタラクション:** 図形をクリックすると補助線が現れる、または角度が表示される等の動的フィードバック。

## 6. アクセシビリティ (Accessibility)
- **コントラスト比:** 数式の読み間違えを防ぐため、記号（+, -, ÷）には十分な白背景の確保とコントラストを設定。
- **クリック領域:** タップターゲットは 44px 四方。
- **配慮事項:** 色盲の生徒が図形の境界を認識できるよう、色だけでなく線種（実線・破線）で判別可能な設計。
