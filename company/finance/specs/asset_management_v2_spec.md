# 💎 資産管理システム Phase 2：製品仕様書 (Final Spec)

## 1. ビジョン
Notionの制約（速度・計算負荷）を突破し、AI-Companyのリポジトリとシームレスに統合された、オーナー専用の**「超高精度・超高速・戦略的財務ダッシュボード」**を構築する。

## 2. 核心的価値 (Core Values)
- **Zero Latency**: 1秒で起動し、即座に入力が反映される。
- **High Fidelity**: 給与明細の項目レベルまで詳細に記録・監査。
- **Proactive Insights**: 過去データとカレンダー予定に基づく、攻めの財務アドバイス。
- **Trust**: 理論値と実測値の不一致を自動検出し、不明金をゼロにする。

## 3. 機能要件

### 3.1 入力システム (Input Layer)
- **Quick Capture**: LINE Inbox経由の即時メモ（現金支出等）。
- **Data Import**: 銀行、PayPay、Suica等のCSV一括インポート（AI名寄せ機能付）。
- **Specialized Input UI**: 給与明細専用フォーム（額面、各種控除、手当を全項目個別入力）。

### 3.2 処理システム (Processing Layer)
- **自動カテゴリ判別**: 過去のパターンから支出項目を自動タグ付け。
- **整合性監査 (Reconciliation)**: キャッシュフローから算出される「論理残高」と、月末入力の「実際残高」の差分計算。
- **給与監査**: 固定された給与規定と照らし合わせ、不適切な控除や不足がないか検証。

### 3.3 出力システム (Output Layer)
- **Interactive Dashboard**: カテゴリ別、月別、年別のグラフィカルな分析。
- **AI Financial Report**: 
  - 月末・中期（15-17日）の自動レポート。
  - 支出予測（来月のカレンダー上のイベント費用を含む）。
  - 無駄遣いの指摘と改善提案。

### 3.4 報酬・ゲーミフィケーション (Reward & Gamification Layer)
- **モチベーション駆動エンジン**: 私生活の努力（AI-Companyでの活動、資格勉強など）を定量化し、自身の「お小遣い（投資・ご褒美資金）」に変換する独自のしくみ。
- **時給換算の排除**: 時間ではなく**「行動（アクション）と成果」**に焦点を当てたポイントシステム（仮称： `ACT = AI-Company Token`）を導入。
  - *案1（ライフ・コミット）*: `.secretary/logs/` の同期セッション（`/sync_session`）完了の度に +10 ACT。
  - *案2（タスク消化力）*: 優先度の高い完了TODO数に応じたポイント付与。
  - *案3（ストリーク）*: 継続的な学習や開発日数へのボーナス加算。
- **トークン還元**: 月末決算時、獲得したACTを日本円（お小遣い）に換算し、次月のご褒美予算として可視化する。


## 4. 非機能要件
- **ポータビリティ**: リポジトリをコピーするだけで全システムが動作する。
- **除外事項**: **時給換算、および労働時間に対する金額評価機能は実装しない。**
- **デザイン**: プレミアムかつモダン、ダークモード対応の洗練されたUI。

## 5. データモデル案
- `master/assets.json`: 資産ごとの最新残高。
- `records/YYYY-MM_expenses.json`: 日々の支出記録。
- `records/salary_slips.md`: 給与明細の詳細ログ。
