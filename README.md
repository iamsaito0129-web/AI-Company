# AI-Company (一人社長のAIチーム - Antigravity)

## 概要
このリポジトリは、Google Antigravity の「マルチエージェント・オーケストレーション」を活用した、自律型AIチームの運用基盤です。
秘書（Secretary）がオーナーの窓口となり、CEOが意思決定を行って専門部署（PM、デザイン、開発、経理等）へ作業を割り振る、組織規模のAI運用を実現します。

## ディレクトリ階層と役割

```
AI-company/
├── .agent/                     # Antigravity用の技術定義コア
│   ├── skills/                 # 各部署・エージェントのスキル定義 (Custom Skills)
│   └── workflows/              # 定型業務の自動化定義
├── company/                    # 実働データ・管理ディレクトリ（全15部署）
│   ├── design/                 # デザイン部：デザイン分析・管理・生成
│   ├── consulting/             # 【新設】コンサル部：外部組織（SAC等）の戦略・体制管理
│   ├── hr/                     # 人事部：AIエージェント管理、RMS
│   ├── engineering/            # 開発部：技術設計、実装管理
│   ├── lifestyle/              # ライフスタイル部：生活・資産管理
│   └── ...                     # 計15部署
├── projects/                   # 実装プロジェクト（ソースコード・実行環境）
│   ├── puzzle-solver/          # 3Dパズル解決アルゴリズム
│   └── ...
├── _archive/                   # 不要ファイルの退避先
└── README.md                   # このファイル
```
