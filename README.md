# AI-Company (一人社長のAIチーム)

## 概要
このリポジトリは、Claude Codeの「ディレクトリ継承メカニズム」を利用した、自律的AIエージェントチームの基盤です。

## 構成部門
- **[CEO/](CEO/CLAUDE.md)**: 戦略的意思決定、リスク管理、RISEフレームワーク
- **[Secretary/](Secretary/CLAUDE.md)**: タスク進行管理(GitOps)、組織記憶
- **[Researcher/](Researcher/CLAUDE.md)**: 市場調査、Deep Research、FICフレームワーク

## ワークスペース
- **01_ABOUT_ME/**: 創業者のプロフィールと嗜好
- **02_PROJECTS/**: 進行中の案件資料とドラフト
- **03_TEMPLATES/**: 品質基準と共通フォーマット
- **04_OUTPUTS/**: 最終成果物の出力場所

## 使用方法
タスクの内容に応じて、ターミナルで各部門のディレクトリへ `cd` 移動し、`claude` (またはAntigravity等のエージェント) を起動してください。そのディレクトリの `CLAUDE.md` に基づいた専門ペルソナがあなたをサポートします。
